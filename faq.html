Freedom FAQ

LINE関連

  1. 友達に追加すると、相手の「知り合いかも？」に表示されるよ。グループトークでの挨拶と併せて、在籍メンバーが不安を感じずに、新規メンバーを追加することができるね(ｏ'д')从('д'ｏ) ﾊｼﾞﾒﾏｼﾃ~
  2. 連続的に友達追加を行うと、追加時にエラーが表示される場合があるから気をつけてね！ アカウントに制限がかかり、24 時間は友達の追加ができなくなるよ。 対策として、20～30 人を追加したら、数十分時間をあけて追加すると安心だね( ｡･ω･｡)ﾉ 凸 ﾎﾟﾁﾎﾟﾁｯﾄﾅ
  3. 新規メンバーが適切に追加をしていても「知り合いかも？」に表示されない場合もまれにあるよ。 追加完了報告後も表示されていない場合は、非表示リストなどに入ってしまっていないか確認しよう。 それでも見つからなければ、新規メンバーに登録漏れがないか確認してみよう！
  4. 追加作業中に、なんらかのエラーメッセージが出た場合、その内容のとおりだよ。 連続追加による制限がかかってしまったか、相手の友達人数が上限（5000 人）に達しているかだよ。 エラーメッセージの内容で判断し、後者の場合は、GT にて該当メンバーに報告してね(｡•̀ω-)b ｵｹ
  5. メンバーを友達追加すると、相手の名前を編集できるよ（名前横のペンマーク） グループごとに同マーキングをしておくと、自分が退会するときなどのメンバー整理がスムーズだよ。 友達追加ができていることの証明にもなるよ＼_(･ω･`) ｺｺﾓﾀﾞｲｼﾞﾈ!!
  6. マーキング（相手の名前を編集）すると、以後、相手が名前を変えても連動表示されないよ。 本来の名前に戻したい場合は、一度、空欄で保存すると可能だよ((φ(･д･｡) ﾜｶｯﾀ ﾒﾓﾒﾓ
  7. メンバーが退会しても、それだけだと自分との「友達」は継続中だよ。 その後、ブロック削除するのか、せずにハートのやり取りをするのかは各自の自由だよ((φ(･д･｡) ﾎｫﾎｫ ﾅﾙﾎﾄﾞﾈ
  8. 退会者が出ていないか確認したいときは、「上記の方」をトーク検索しよう。 グループ管理方法の都合上、見つけやすいよ(｡･ω･｡) ﾌﾑﾌﾑ
  9. 在籍メンバーをブロック削除したいときは、自分の友達一覧から操作してね！ グループメンバー一覧から削除すると退会してもらうことがあるよ！=͟͟͞͞(๑•̀д•́๑=͟͟͞͞(๑•̀д•́๑=͟͟͞͞(๑•̀д•́๑) ﾅ､ﾅﾝﾃｺｯﾀｧ!!!!
  10. グループを退会する際、自ら退会処理をすると、グループの過去ログがすべて見られなくなるよ(((`･ω･)ゞ ﾘｮ!
  11. 将来的に再入会を希望しているなど、なんらかの都合で、退会後も過去ログを確認したいなら、他メンバーに退会処理を依頼しよう。気軽に管理メンバーに申し出てね(*´꒳`*) もちろん、自分が退会後のログ進行は見られないよ( •ω•｡)و ﾜｶﾘﾏｼﾀｰ

グループ関連

  1. 再入会を希望する場合、管理メンバーに連絡してね。 退会経緯によっては、お断りすることもあるよ！ 管理メンバー側がブロック削除している場合は、あなたの個チャは届かないよ、ごめんね！ 再入会の可能性もある旨、伝えてから退会すれば、管理メンバーはブロック削除しないよ（たぶんね！）(･д･｡) ｵｹｵｹ
  2. 突発的な事故等により、LINE アカウントデータにアクセスすること自体が出来なくなり、その他の方法 でもグループに繋がるなにかを探しだせない場合は、縁がなかったと諦めようｗ リスクがあるため、管理メンバーが LINE ID 検索などを、常にオープンにしておく保証はないよ(TωT)/~~~ ｺﾞﾒﾝﾈ
  3. ブロックすると制限のある「友達」になり、もろもろの個人的な表示がされなくなるよ。 解除したい場合は、ブロックリストから操作しよう( • ω•́ )ゞ ｶｼｺﾏﾘｯ!
  4. ブロックリストから削除すると、「友達」関係が解除されるよ。 その状態からブロックを解除するには、誰かから該当アカウントの連絡先を知らせてもらわない限り無理だよ。同グループに双方が在籍しているなら、もちろんその限りではないね！(｡•ㅅ•｡) ﾅﾙﾎﾄﾞﾈ

ツムツム関連

  1. LINE で友達追加しても、即時反映はされないよ。 ツムツムの表示に反映されるまで、数分～半日ていどのタイムラグがあるよ(ºωº;≡;ºωº ) ﾁｮｯﾄﾏｯﾃﾃﾈ
  2. 上記、タイムラグ間に、グループメンバーがツムツムの友達招待リストに載っている可能性がある よ。その場合は、在籍メンバー新規メンバー、ともに「友達招待」が可能だよ。 ツムツム内のことなので、グループルールとしても禁止していないよ ((o(*^∇^)X(^∇^*)o)) ﾔｯﾀﾈ!
  3. ハートの送受信方法に、グループルールとしての指定はないよ。 ツムツムのシステムとして、メールボックスから一括で受け取ると、相手に返送はされないよ。 ランキングから送信するか、メールボックスから個別で受け取ることで、相手に送信されるよ(v〃ω〃)八(〃ω〃v) ﾊｰﾄｺｳｶﾝ､ｱﾘｶﾞﾄﾈ~!
  4. 同じ相手に送信できるハート数は、一時間に一つだよ。 例えば、ランキングから送信した直後に、メールボックスから個別で受け取っても、送られているハート数は一つということだよヾ(｡º∀º)ﾉ━♡ ﾄﾄﾞｷﾏｽﾖｳﾆ
  5. 自分が友達登録していない相手や、ブロック削除した友達から受信したハートは、「unknown」表示だよ。
